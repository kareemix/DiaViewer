![image](https://kareemix.github.io/assets/diaviewerex.png)

# Dependencies
* shiny
* arrow
* ggplot2
* reticulate
* duckdb
* duckplyr

# Instructions
* Create directory data/ and move .xic.parquet files into it
* Run ```./runtime``` in your terminal
* Open http://localhost:3030/ in your browser
